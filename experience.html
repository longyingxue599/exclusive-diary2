我的经验

学习经历

7天学习计划

Day 1-3：掌握网页的皮囊（HTML/CSS）。
Day 4：理解网页的灵魂（JS 数据渲染）。
Day 5-6：构建网页的大脑与记忆（Node + Database）。
Day 7：完成云端进化（Deployment）。

核心软件工具清单 (必备)

编辑器：VS Code（前端开发标准配置，安装插件如 Auto Close Tag, Prettier）。
浏览器：Chrome（重点学习使用其 F12 开发者工具 进行调试）。
版本管理：Git（配合 GitHub Desktop 或命令行使用，用于代码托管）。
运行环境：Node.js（安装 LTS 版本，自带 npm 包管理器）。
API 测试：Postman 或 VS Code 插件 Thunder Client（用于测试你的 Node.js 后端接口）。

学习与实战建议顺序

第一阶段：网页基石 (1-2周) 掌握 HTML 语义化标签、CSS 盒子模型、Flexbox/Grid 布局。
实战：用纯 HTML/CSS 仿写一个静态的博客首页 。
第二阶段：交互进阶 (2-4周)
学习 JS 语法（变量、函数、ES6+ 新特性）、DOM 操作。
实战：给静态页面加上点击跳转、暗黑模式切换等功能。
第三阶段：工程化与部署 (1周)
学习 Git 基础命令（add, commit, push）。
实战：将你的静态页面上传至 GitHub Pages 预览。
第四阶段：后端与数据库 (你的目标核心) (2-3周)
Node.js (Express)：学习如何写一个简单的 API 接口。
Supabase (PostgreSQL)：学习如何在 Supabase 创建表，利用其提供的 JS SDK 进行 CRUD（增删改查）操作。
免费部署：后端可以尝试部署在 Render 或 Vercel 的免费层级。。
