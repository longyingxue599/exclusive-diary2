HTML 属性快速复习表（A4打印版）

每个属性包含：作用 / 核心功能 / 视觉效果提示 / 设计目的。符号帮助联想记忆。

结构 Structure

符号属性作用 / 核心功能视觉效果提示设计目的
🔗id元素唯一标识无直接视觉效果便于 JS 操作、样式定位
⚓class元素分类标识可通过 CSS 样式化批量样式、复用设计
📐style内联样式定义可直接影响布局、颜色等快速调试或单独样式
📦hidden隐藏元素元素不显示控制页面显示/逻辑
🖇️data-*自定义数据属性无视觉变化存储 JS 数据 / 状态

文本 Text

符号属性作用 / 核心功能视觉效果提示设计目的
📝title提示信息鼠标悬停显示气泡提供额外信息 / 辅助说明
✍️lang指定语言可影响拼写检查/朗读国际化 / 辅助工具优化
💬dir文本方向LTR / RTL 显示变化适配不同语言文字排版

表单 Form

符号属性作用 / 核心功能视觉效果提示设计目的
🔑name表单字段名无视觉变化表单提交数据标识
✉️value表单默认值输入框显示默认内容预填充 / 提示用户
📌placeholder输入提示文本灰色提示文字引导用户填写内容
✅required必填字段表单提交提示保证用户输入必填信息
🔒disabled禁用控件灰色不可操作限制用户操作，避免误操作

媒体 Media

符号属性作用 / 核心功能视觉效果提示设计目的
🎞️src资源路径显示图像 / 视频 / 音频加载并展示媒体内容
⏯️autoplay自动播放媒体自动开始播放提升用户体验 / 动态感
🔇muted静音无声音初始静音，防止打扰
🔄loop循环播放媒体反复播放持续动画 / 背景效果
📏width / height尺寸控制图像 / 视频大小变化布局适配 / 保持比例

⚡rel链接关系SEO 或安全提示优化安全、搜索引擎友好
