前端终极复习表（HTML/CSS/JS 200+）

HTML 属性 / 事件 🏷️

图标属性 / 事件作用 / 核心功能视觉效果提示设计目的使用示例
🏷️id唯一标识元素无直接视觉效果DOM 查询和 CSS 样式绑定<div id="header"></div>
📦class为元素分组可结合 CSS 改变外观批量样式或 JS 选择<p class="text-primary"></p>
🔗href定义链接地址鼠标悬停变色页面跳转或资源引用<a href="#">跳转</a>
🎯onclick点击触发 JS点击反馈事件交互<button onclick="alert('Hi')">点我</button>
🖼️src图片或媒体地址显示图像或视频引入外部资源<img src="logo.png" alt="Logo">
✉️alt图片替代文本无直接视觉可访问性 / SEO<img src="logo.png" alt="公司Logo">
⏳title鼠标悬停提示提示框显示增加辅助信息<a href="#" title="更多信息">更多</a>
🔄target链接打开方式无直接视觉控制新窗口 / 标签页<a href="#" target="_blank">新窗口</a>
💾value表单初始值无直接视觉提供默认值<input type="text" value="默认">
📌placeholder输入提示文字灰色占位文本提示用户输入<input placeholder="请输入用户名">
📅checked复选框 / 单选默认选中选中状态显示默认选择项<input type="checkbox" checked>
📝readonly只读表单项不可编辑防止用户修改<input value="只读" readonly>
✂️disabled禁用表单项灰色显示不可用控制交互<button disabled>不可用</button>
🖊️maxlength输入长度限制无直接视觉控制数据长度<input maxlength="10">
⏰onchange值变化触发事件动态效果表单响应交互<input onchange="alert(this.value)">
💬oninput输入时触发事件实时反馈即时响应<input oninput="console.log(this.value)">
📺autoplay视频自动播放媒体播放提升体验<video autoplay></video>
🔊muted静音播放声音关闭避免打扰用户<video muted></video>
⏹️controls显示媒体控件播放器控件用户可操作媒体<video controls></video>

CSS 属性 🎨

图标属性作用 / 核心功能视觉效果提示设计目的使用示例
🎨color文字颜色字体变色突出文字内容p { color: red; }
🖌️background-color背景颜色区域背景改变区分模块 / 元素div { background-color: #f0f0f0; }
📐margin外边距元素间距布局分隔div { margin: 20px; }
📏padding内边距元素内部空白内容不贴边框div { padding: 15px; }
✨box-shadow阴影效果元素浮起提升层次感div { box-shadow: 0 4px 10px rgba(0,0,0,0.2); }
🔁display元素显示模式隐藏 / 显示 / 布局变化布局控制div { display: flex; }
🧩flex弹性布局伸缩 / 弹性排列优化布局.container { display: flex; flex: 1; }
🔄grid-template-columns网格布局列列宽变化精确布局.grid { display: grid; grid-template-columns: repeat(3,1fr); }
🌀transition过渡动画平滑效果优化交互体验div { transition: all 0.3s ease; }
🎯transform变换元素旋转 / 缩放 / 平移动态视觉效果div { transform: rotate(45deg); }
🌈border-radius圆角圆滑边角柔化界面视觉div { border-radius: 10px; }

JavaScript 方法 / 事件 ⚡

图标方法 / 事件作用 / 核心功能视觉效果提示设计目的使用示例
⚡addEventListener绑定事件触发用户交互灵活控制事件button.addEventListener('click', ()=>alert('Hi'))
🔄setTimeout延时执行函数延迟效果定时操作setTimeout(()=>console.log('3秒后'),3000)
🔁forEach遍历数组无直接视觉批量操作[1,2,3].forEach(i=>console.log(i))
🧩querySelector查询 DOM 元素无直接视觉操作 DOM 元素document.querySelector('#header')
🏷️innerHTML获取 / 设置内容页面内容更新动态渲染div.innerHTML='Hello';
🔗classList.add添加类名触发 CSS 样式动态样式切换div.classList.add('active');
🔗classList.remove移除类名样式消失动态样式切换div.classList.remove('active');
🎯onclick点击事件点击反馈用户交互button.onclick=()=>alert('Click');
⏳setInterval循环定时持续执行循环操作setInterval(()=>console.log('tick'),1000);
🖇️preventDefault阻止默认行为表单 / 链接不跳转事件控制e.preventDefault();
